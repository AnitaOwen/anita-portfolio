About Me

I'm Anita Owen, a Full Stack Developer based in New York City, passionate about creating meaningful and impactful solutions through technology. I thrive on solving complex problems and continuously learning new skills. My experience spans full-stack development, with expertise in the PERN stack (PostgreSQL, Express, React, and Node.js), and I’ve built projects like FridgeM8, a tool designed to help reduce food waste, and Global Agent, a gamified educational platform. I’m driven by curiosity, a love for both logic and creativity, and a deep commitment to building projects that make a difference. I look forward to growing further as a developer and contributing to innovative, impactful solutions.

Projects

  • Global Agent

    Global Agent is a gamified educational platform that engages middle and high school students by challenging them to solve cases based on real-world global events. As a key member of a five-person team, I led backend development efforts, co-architecting the server infrastructure. I integrated Google Cloud Translate for real-time news translation and implemented AI-powered question generation, enabling the creation of dynamic, multiple-choice quizzes that enhance student learning and engagement.

  • FridgeM8

    FridgeM8 is a web application built to help users track expiration dates on fridge items, minimize food waste, and promote smarter spending decisions. I designed an intuitive user interface and implemented features for tracking expiration dates, backed by a PostgreSQL database for efficient data management. Using the PERN stack (PostgreSQL, Express, React, and Node.js), I gained valuable experience in full-stack development, focusing on performance optimization and seamless user experience.

    Live Demo Backend Repo Frontend Repo

  • Hidden Murals NYC

    Hidden Murals NYC is a web application that enables users to discover and interact with hidden murals throughout New York City. As part of a collaborative team, I contributed to both front-end and back-end development, implementing full CRUD functionality. Key features include mural submissions with Cloudinary image uploads, search and filter capabilities by borough, and detailed views for individual murals. The application is fully responsive, ensuring an optimal user experience across devices.
